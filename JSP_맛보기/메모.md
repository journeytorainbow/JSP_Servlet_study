# JSP 맛보기

+ `JSP` : `HTML` 안에 `JAVA`코드를 넣음.

+ 웹 컨테이너 구조


    + `JSP`는 **웹 컨테이너**에서 컴파일되는 ***동적인***파일이다.
        + 사용자에게 `JSP` 문법은 보여지지 않고 `HTML`로 보여짐(소스보기로 확인 가능).

+ `JSP`파일은 프로젝트의 `WebContent`디렉토리 하위에 위치한다.
